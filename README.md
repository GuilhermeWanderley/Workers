# Sistema de Gestão de Trabalhadores (Java)

Projeto desenvolvido em Java com foco na prática de Programação Orientada a Objetos (POO), simulando o gerenciamento de trabalhadores, departamentos e contratos por hora.

## Funcionalidades
- Cadastro de trabalhador
- Associação com departamento
- Registro de múltiplos contratos
- Cálculo da renda mensal com base nos contratos

## Conceitos aplicados
- Programação Orientada a Objetos (POO)
- Encapsulamento
- Herança e polimorfismo
- Relacionamento entre classes (composição)
- Estruturas de repetição (for)
- Manipulação de listas

## Tecnologias utilizadas
- Java
